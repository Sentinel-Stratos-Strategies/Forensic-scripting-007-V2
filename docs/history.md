# Project History

Forensic Scripting 007 V2 is a cleaned, reusable successor to earlier private
007 forensic tooling. The V2 repository is intentionally product-safe: it keeps
reusable code, schemas, tests, CI, and documentation while leaving private case
evidence, local run outputs, old branch archaeology, and internal review exports
outside the public repository.

The practical boundary is simple:

- V2 contains the reusable toolkit.
- Private investigations and evidence packages stay outside git.
- Historical review material is kept locally by maintainers when needed, not
  published as product documentation.
